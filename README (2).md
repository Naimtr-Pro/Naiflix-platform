# 🎬 Naiflix Platform

Plataforma de streaming inspirada na Netflix, desenvolvida por **Naiane** durante a *Imersão Front-End com IA* da Alura.

O projeto explora conceitos fundamentais de desenvolvimento web, utilizando **HTML, CSS e JavaScript**, com foco em boas práticas de interface e experiência do usuário, simulando um ambiente real de streaming.

---

## 🌐 Acesse o projeto

Você pode visualizar a aplicação online através do link:

👉 https://naimtr-pro.github.io/Naiflix-platform/

---

## 🧩 Objetivo

Desenvolver uma interface **funcional, moderna e responsiva** que simula uma plataforma de streaming, incluindo:

- Organização de conteúdos por categorias  
- Carrosséis de filmes e séries  
- Identidade visual inspirada em serviços reais  

---

## 🛠 Tecnologias utilizadas

- **HTML5** — estrutura da aplicação  
- **CSS3** — estilização e responsividade  
- **JavaScript** — interatividade e manipulação dos carrosséis  
- **Google Fonts** — tipografia personalizada  
- **Ferramentas de IA** — apoio no desenvolvimento  

---

## 📺 Funcionalidades

- 🎥 Layout inspirado na Netflix  
- 📂 Organização por categorias  
- 🎞 Carrosséis horizontais interativos  
- 🖼 Cards com imagens e títulos  
- 🧭 Navegação simples e intuitiva  
- 📱 Design responsivo (desktop e mobile)

## 🚀 Como executar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/naimtr-pro/Naiflix-platform.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Naiflix-platform
   ```

3. Abra o arquivo `index.html` no navegador.

> ⚠️ Importante: mantenha a estrutura de pastas original para garantir o correto carregamento de imagens e estilos.
