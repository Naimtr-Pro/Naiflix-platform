🎬 Naiflix Platform
Plataforma de streaming inspirada na Netflix, desenvolvida por Naiane durante a Imersão Front-End com IA da Alura. O projeto explora conceitos de HTML, CSS, JavaScript e design responsivo para criar uma experiência moderna e intuitiva.
🌐 Acesso ao projeto
👉 https://naimtr-pro.github.io/Naiflix-platform/
🧩 Objetivo
Criar uma interface funcional que simula uma plataforma de streaming, com carrosséis de filmes, categorias e uma identidade visual própria.
🛠 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Google Fonts
- Ferramentas de IA como apoio no desenvolvimento
📺 Funcionalidades
- Layout inspirado na Netflix
- Carrosséis horizontais de filmes e séries
- Cards com imagens e títulos
- Navegação simples e intuitiva
- Design responsivo para diferentes telas
🚀 Como executar
Basta abrir o arquivo index.html no navegador após baixar ou clonar o repositório.

Projeto desenvolvido por Naiane durante a Imersão Front-End com IA — Alura.
