
🎬 Naiflix Platform
Plataforma de streaming inspirada na Netflix, desenvolvida por Naiane durante a Imersão Front-End com IA da Alura. O projeto explora conceitos de HTML, CSS, JavaScript e boas práticas de interface, criando uma experiência visual semelhante a serviços reais de streaming.


🌐 Acesso ao projeto
A versão publicada está disponível em:

👉 https://naimtr-pro.github.io/Naiflix-platform/


🧩 Objetivo do projeto

Criar uma interface funcional e responsiva que simula uma plataforma de streaming, com carrosséis de filmes, organização por categorias e uma identidade visual própria.


🛠 Tecnologias utilizadas

- HTML5 para estruturação das páginas
- CSS3 para estilização e responsividade
- JavaScript para interações e carrosséis
- Google Fonts para tipografia
- Ferramentas de IA como apoio no desenvolvimento

  
📺 Funcionalidades principais

- Layout inspirado na Netflix
- Carrosséis horizontais de filmes e séries
- Cards com imagens e títulos
- Navegação simples e intuitiva
- Design responsivo para diferentes tamanhos de tela

🚀 Como executar localmente

- Baixe ou clone o repositório.
- Abra o arquivo index.html no navegador.
- Certifique-se de manter a estrutura de pastas para que imagens e estilos carreguem corretamente.
